# 랭턴의 개미 (Langton's Ant) — 2규칙 튜링 기계

## 무엇을 만드나
격자 위 개미가 단 두 규칙으로 움직이며 칸 색을 뒤집는다. 처음 ~1만 스텝은 혼돈처럼 보이다가, 갑자기 규칙적인 '고속도로' 패턴으로 질서가 창발한다.

## 핵심 개념 — 국소 규칙, 전역 창발
```
현재 칸이 흰색이면:  오른쪽으로 90° 회전, 칸을 검게, 한 칸 전진
현재 칸이 검은색이면: 왼쪽으로 90° 회전, 칸을 희게, 한 칸 전진
```
회전은 방향벡터 `(dx,dy)`의 90° 변환:
- 우회전: `(dx,dy) → (−dy, dx)`
- 좌회전: `(dx,dy) → (dy, −dx)`

## 알고리즘 (의사코드)
```
개미 위치·방향 초기화
repeat 스텝:
    if grid[pos]==0: 우회전; grid[pos]=1
    else:            좌회전; grid[pos]=0
    (그 칸 다시 그림)
    pos += 방향 (토러스 wrap)
```

## 주요 파라미터
- 격자 250², 프레임당 300스텝. 약 10,000스텝 후 '고속도로' 출현.

## 다른 언어로 옮길 때
- 배열 + 방향벡터 회전이면 끝. 매우 이식성 높음.
- 다색·다규칙(Turmite)으로 확장하면 다양한 창발 패턴.
- 이 개미는 튜링 완전 — 단순 규칙의 계산 보편성 예시.
