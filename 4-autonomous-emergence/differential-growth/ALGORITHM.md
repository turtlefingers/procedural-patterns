# 차분 성장 (Differential Growth)

## 무엇을 만드나
노드로 이뤄진 폐곡선이, 부분마다 다른 속도로 길어지며 접히고 주름지는 형태(뇌 주름·산호·케일 가장자리)를 만든다.

## 핵심 개념 — 세 힘 + 성장
각 노드에 세 가지 힘이 작용(Boids의 분리/정렬/결속에 대응):
- **척력(repulsion)**: 가까운 다른 노드를 밀어냄(자기교차 방지·부피감).
- **인력(spring)**: 곡선상 이웃 두 노드와 목표 간격(REST)을 유지.
- **정렬(alignment)**: 이웃 두 노드의 중점 쪽으로 끌어 곡선을 매끄럽게.

그리고 **성장**: 간선이 너무 길어지면 중점에 새 노드를 삽입(길이가 늘어남). 노드가 늘수록 곡선이 담을 공간이 부족해져 좌굴(buckling)·주름이 생긴다.

## 알고리즘 (의사코드)
```
매 스텝:
    각 노드 힘 = 0
    // 척력: 반경 REP_R 내 다른 노드마다 밀어냄
    for i<j: if dist<REP_R: 서로 반대로 밀기(세기 ∝ (REP_R-d)/REP_R)
    // 인력+정렬: 곡선 이웃(prev,next)에 대해
    for i: 스프링((d-REST)/d·SPRING) + 중점끌기(ALIGN)
    각 노드 이동 = 힘·계수 + 미세 랜덤(좌굴 유발), 화면 안으로 클램프
    // 성장
    긴 간선(>MAX_LEN)의 중점에 노드 삽입
    무작위 간선 몇 개에도 노드 주입(면적 증가 압력)
곡선을 부드럽게(스플라인) 그림
```

## 주요 파라미터
- `REP_R=26, SPRING=0.36, ALIGN=0.28, REST=6, MAX_LEN=10, MAX_N=1300`.

## 다른 언어로 옮길 때
- 필요한 것: 점 리스트(가운데 삽입 가능한), 벡터 연산, 곡선 그리기.
- 척력이 O(n²) — 노드가 많으면 공간 격자로 근접만 계산.
- 미세 요동(랜덤)이 있어야 대칭이 깨지고 주름이 생긴다. Boids와의 개념적 대응을 기억.
