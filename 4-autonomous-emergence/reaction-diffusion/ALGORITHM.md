# 반응확산 (Reaction–Diffusion) — Gray–Scott 모델

> **[AI Implementation Context]**
> 이 문서는 해당 알고리즘을 다른 프로그래밍 언어(C++, Python, Unity, Rust 등)나 그래픽스 환경으로 이식(Porting)할 때 참조하기 위한 기하학/알고리즘 명세서입니다. 특정 언어의 그래픽스 API에 종속되지 않은 순수 논리와 수학적 규칙을 담고 있습니다.


## 1. 개요 및 목적 (Overview)
두 화학물질 A·B가 반응하며 서로 다른 속도로 확산하면, 균일한 상태가 스스로 무너져 얼룩·줄무늬·점(튜링 패턴)이 자발적으로 솟는다. 세 파라미터 프리셋(산호/유사분열/솔리톤)을 3개 띠로 동시 시뮬레이션.

## 2. 핵심 이론 및 원리 (Core Concept & Math)
각 셀에서 매 스텝:
```
A' = A + dA·∇²A − A·B² + f·(1 − A)      // A: 공급자, f로 보충
B' = B + dB·∇²B + A·B² − (k + f)·B       // B: 자기촉매(A를 소비해 증식), k로 제거
```
- `A·B²`: B가 A를 잡아먹어 B로 바뀌는 반응(자기촉매).
- **확산 계수 차이**(dA=1.0 > dB=0.5)가 결정적: 억제자(A)가 빨리 퍼지고 활성자(B)가 느리게 퍼져야 패턴이 생긴다.
- `∇²`(라플라시안)은 3×3 가중 합(중앙 −1, 상하좌우 0.2, 대각 0.05).

## 3. 알고리즘 의사코드 (Pseudocode)
```
A = 1로 채움, B = 0;  몇 군데 B=1 씨앗
매 스텝(여러 번 반복):
    for each cell:
        lapA = 3×3 가중합(A);  lapB = 3×3 가중합(B)
        abb = A·B·B
        A2 = A + dA·lapA − abb + f·(1−A)
        B2 = B + dB·lapB + abb − (k+f)·B
    swap(A,A2), swap(B,B2)      // 더블 버퍼(동시 갱신)
렌더: 밝기 = A − B
```

### 주요 파라미터 (Main Parameters)
- 프리셋 (f,k): 산호(0.0545, 0.062), 유사분열(0.0367, 0.0649), 솔리톤(0.030, 0.060).
- `dA=1.0, dB=0.5`. f,k 조금만 바꿔도 무늬가 완전히 달라진다.

## 4. 데이터 구조 및 이식 가이드 (Data Structures & Porting Guide)
- 필요한 것: 부동소수 배열 두 쌍(A,B 더블버퍼), 픽셀 렌더.
- **동시 갱신**(새 배열에 쓰기)과 경계 wrap이 필수. 안정성 위해 스텝당 여러 번 반복.
- 큰 격자는 셰이더로. f,k 지도를 공간적으로 바꾸면 무늬가 지역마다 달라짐.
