# 델로네 삼각분할 (Delaunay Triangulation) — Bowyer–Watson

## 무엇을 만드나
평면의 점들을, 어떤 삼각형의 **외접원 안에도 다른 점이 없도록** 삼각형으로 잇는다(가장 "살찐" 삼각형들). `V`로 그 쌍대인 보로노이 경계를 겹쳐 볼 수 있다.

## 핵심 개념 — Bowyer–Watson 증분 삽입
점을 하나씩 추가하며 삼각망을 갱신:
1. 새 점의 **외접원이 그 점을 포함하는** 삼각형("나쁜 삼각형")들을 찾는다.
2. 그 삼각형들을 지우면 다각형 구멍(경계 간선들)이 생긴다.
3. 새 점과 각 경계 간선을 이어 새 삼각형으로 메운다.

외접원 계산(외심 ux,uy와 반지름²)은 세 점 좌표의 행렬식 공식으로.

## 알고리즘 (의사코드)
```
T = [아주 큰 슈퍼삼각형(모든 점 포함)]
for p in points:
    bad = { t in T : p 가 t의 외접원 안 }
    boundary = bad들의 간선 중 정확히 한 번만 등장하는 것(공유되지 않은 간선)
    T -= bad
    for e in boundary: T += triangle(e.a, e.b, p)
슈퍼삼각형 꼭짓점을 포함한 삼각형 제거
```
보로노이 오버레이: **인접(간선 공유) 삼각형들의 외심을 잇는다**(외심 = 보로노이 정점).

## 주요 파라미터
- 점 60개. 외접원 판정 부동소수 오차에 주의(`|det|<1e-9` 예외처리).

## 다른 언어로 옮길 때
- 순수 기하(행렬식, 외접원)만 필요. O(n²) 단순판. 대규모는 라이브러리(delaunator) 권장.
- 델로네와 보로노이는 **쌍대** — 하나를 만들면 다른 하나가 공짜(외심을 이으면 보로노이).
