# 푸아송 디스크 샘플링 (Poisson-Disk) — Bridson 알고리즘

## 무엇을 만드나
서로 **최소 거리 R 이상** 떨어졌지만 무작위로 흩뿌려진 점들(블루 노이즈). 뭉침·격자 정렬이 없어 자연스럽다(망막 광수용체 배열과 유사).

## 핵심 개념 — Bridson (2007), O(n)
- **배경 격자**: 셀 크기 `R/√2`. 그러면 한 셀에 점이 최대 하나 → 근접 검사가 상수 시간.
- **활성 목록(active list)**: 아직 주변에 점을 더 놓을 수 있는 점들.

## 알고리즘 (의사코드)
```
격자 준비(셀 = R/√2);  시작점 하나 삽입(활성)
while 활성 목록이 있음:
    base = 활성 목록에서 무작위 하나
    k(=30)번 시도:
        후보 = base 주변 [R, 2R] 반지름의 랜덤 점
        if 후보가 화면 안 && 주변 (5×5 셀 내) 모든 점과 거리 ≥ R:
            후보 삽입(격자+활성);  성공 → 다음 while
    k번 다 실패하면 base 를 활성에서 제거(주변이 꽉 참)
```

## 주요 파라미터
- `R=18`(최소 간격), `K=30`(재시도 횟수).

## 다른 언어로 옮길 때
- 필요한 것: 2D 배열(격자), 거리 계산, 난수. 그래픽 무관.
- 핵심 최적화는 **격자로 근접 점만 검사**하는 것 — 이게 O(n)의 비결.
- 이 점들은 voronoi/delaunay의 좋은 입력이 된다(축끼리 맞물림).
