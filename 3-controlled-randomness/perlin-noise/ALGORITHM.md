# 펄린 노이즈 (Perlin Noise)

> **[AI Implementation Context]**
> 이 문서는 해당 알고리즘을 다른 프로그래밍 언어(C++, Python, Unity, Rust 등)나 그래픽스 환경으로 이식(Porting)할 때 참조하기 위한 기하학/알고리즘 명세서입니다. 특정 언어의 그래픽스 API에 종속되지 않은 순수 논리와 수학적 규칙을 담고 있습니다.


## 1. 개요 및 목적 (Overview)
격자 기반의 **부드럽고 일관된 무작위성** 필드. 인접 값이 매끄럽게 이어져 지형·구름·불·대리석의 자연스러운 질감을 만든다. 여기선 2D 필드를 시간축(z)으로 흘려 애니메이션.

## 2. 핵심 이론 및 원리 (Core Concept & Math)
격자 꼭짓점마다 무작위 **기울기 벡터**를 놓고, 임의의 점에서는 주변 네 꼭짓점의 기여를 **부드러운 보간(fade curve)** 으로 섞는다. 값 노이즈보다 방향성이 자연스럽다.
- 페이드 함수: `6t⁵−15t⁴+10t³`
- **옥타브(octave)**: 주파수를 배로 올리며 진폭을 반으로 줄여 여러 노이즈를 합침(프랙탈 노이즈) → 큰 굴곡+작은 디테일. `noiseDetail(octaves, 0.5)`가 이 역할.

## 3. 알고리즘 의사코드 (Pseudocode)
```
매 프레임 z += 0.008
for y, x in 저해상 격자:
    v = noise(x·s, y·s, z)      // 0..1
    픽셀 밝기 = v·255
저해상 이미지를 화면 크기로 확대
```
(스케치는 p5 내장 `noise()`를 씀. 직접 구현하려면 아래.)

### 순수 구현 스케치
```
grad[gx][gy] = 단위 무작위 벡터
noise(x,y):
    셀 좌표 gx,gy = floor;  국소좌표 fx,fy = x-gx, y-gy
    u,v = fade(fx), fade(fy)
    각 모서리의 (기울기·거리) 내적 4개를 u,v로 이중선형 보간
```

## 4. 데이터 구조 및 이식 가이드 (Data Structures & Porting Guide)
- 내장 노이즈가 없으면 위 순수 구현 필요(기울기 격자 + fade + 보간). 심플렉스 노이즈도 대안.
- 옥타브 합성(fBm)으로 디테일 조절. 시드를 고정하면 재현 가능.
