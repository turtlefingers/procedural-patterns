# 잎차례 (Phyllotaxis) — Vogel 모델

## 1. 개요 및 목적 (Overview)
해바라기 씨앗처럼, 점들을 중심에서 바깥으로 **황금각(≈137.5°)** 씩 돌리며 배치하면 겹침 없이 촘촘한 이중 나선이 나타난다.

## 2. 핵심 이론 및 원리 (Core Concept & Math)
n번째 점의 극좌표:
```
θ = n · 137.5077…°        // 발산각(divergence angle) = 황금각
r = c · √n                // 중심에서의 거리 (√n 이라 밀도가 균일)
(x, y) = (r·cosθ, r·sinθ)
```
황금각을 쓰는 이유: 황금비 φ가 "가장 무리한 수"라 어떤 분수로도 잘 근사되지 않아 점들이 정렬(줄맞춤)되지 않고 최적으로 흩어진다. 각을 137.5°에서 조금만(±0.1°) 벗어나게 하면 나선이 방사형 살로 무너진다(스케치에서 ←/→로 실험).

## 3. 알고리즘 의사코드 (Pseudocode)
```
for n = 0, 1, 2, ...:
    r = C · sqrt(n)
    if r > 최대반경: break
    θ = n · radians(137.5077)
    점 (cx + r·cosθ, cy + r·sinθ), 크기는 r에 비례
```

## 4. 데이터 구조 및 이식 가이드 (Data Structures & Porting Guide)
- 필요한 것: `sqrt`, `cos/sin`, 원 그리기. 극히 이식성 높음.
- 발산각을 변수로 두면 137.5°의 특별함을 시연할 수 있다.
- 큰 각을 정확히: `137.5077640500378°` 사용.
