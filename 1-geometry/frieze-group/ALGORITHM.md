# 프리즈군 (Frieze Groups)

## 무엇을 만드나
하나의 모티프(motif, 여기서는 깃발 모양)를 **한 방향의 띠(band)** 를 따라 반복시킬 때 나올 수 있는 대칭 유형은 수학적으로 **정확히 7가지**뿐이다. 이 스케치는 7종(p1, p11g, p1m1, p11m, p2, p2mg, p2mm)을 7개의 가로 띠로 나란히 보여준다.

## 핵심 개념
프리즈군은 "1차원 병진(translation) 대칭 + 아래 4가지 대칭의 조합"으로 완전히 분류된다.

- **T** : 좌우 병진(항상 존재)
- **V** : 수직 거울(반사) — 좌우 뒤집기 `scale(-1, 1)`
- **H** : 수평 거울(반사) — 상하 뒤집기 `scale(1, -1)`
- **R** : 180° 회전 = 좌우+상하 동시 뒤집기 `scale(-1, -1)`
- **G** : 미끄럼 반사(glide) — 반 칸 이동 후 상하 뒤집기

7종은 이 요소들을 어떻게 켜고 끄느냐의 조합이다.

## 알고리즘 (언어 독립 의사코드)
```
motif()  // 원점 기준으로 그리는 기본 무늬 하나 (선/삼각형/원 등)

place(x, sx, sy):          // (x,0)로 이동 후 (sx,sy)로 스케일하여 모티프 그림
    push_transform()
    translate(x, 0)
    scale(sx, sy)          // sx,sy ∈ {+1,-1} 로 반사/회전을 표현
    motif()
    pop_transform()

// 7개 그룹 = 한 주기(period P) 안에 place를 어떻게 배치하는가
groups = [
  p1   : place(x, +1,+1)                                             // T만
  p11g : place(x, +1,+1); place(x+P/2, +1,-1)                        // 글라이드
  p1m1 : place(x, +1,+1); place(x, -1,+1)                           // 수직거울
  p11m : place(x, +1,+1); place(x, +1,-1)                           // 수평거울
  p2   : place(x, +1,+1); place(x+P/2, -1,-1)                        // 180°회전
  p2mg : place(x,+1,+1); place(x,-1,+1); place(x+P/2,+1,-1); place(x+P/2,-1,-1)
  p2mm : place(x,+1,+1); place(x,-1,+1); place(x,+1,-1); place(x,-1,-1)
]

for each group row:
    for x = start to width step P:
        group(x)
```

## 주요 파라미터
- `P = 60` : 반복 주기(모티프 간 간격)
- `ROW_GAP = 68` : 7개 띠의 세로 간격
- 모티프는 좌우/상하 비대칭이어야 각 대칭 유형이 시각적으로 구분된다(대칭 모티프를 쓰면 7종이 똑같아 보임).

## 다른 언어로 옮길 때
필요한 것은 **2D 아핀 변환 스택**(translate/scale/push/pop)뿐이다. 없으면 각 꼭짓점에 `x' = ±x, y' = ±y` 부호를 직접 곱해도 된다. 반사는 음수 스케일로 표현된다는 점이 핵심.
