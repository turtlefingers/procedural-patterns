# 만화경 (Kaleidoscope) — 방사 대칭 드로잉

## 무엇을 만드나
마우스로 그린 획 하나를, 중심을 기준으로 **n개의 회전 복제 + 거울 반사**로 즉시 복제해 만화경 무늬를 만든다. 입력은 사용자의 자유로운 드래그, 출력은 대칭 문양.

## 핵심 개념
방사 대칭군 **D_n (dihedral group)**: n중 회전(2π/n) + 반사. 사용자가 그리는 선분 하나 `(pmouse → mouse)`를 다음처럼 2n번 복제한다.
- n번의 회전: 각도 `i·2π/n`
- 각 회전마다 거울 반사 1번 추가(`scale(1,-1)`) → 총 2n개

## 알고리즘 (의사코드)
```
매 프레임, 마우스가 눌린 동안:
    translate(center)
    p_prev = 직전 마우스 좌표 - center
    p_cur  = 현재 마우스 좌표 - center
    for i in 0..n-1:
        push()
        rotate(i * 2π / n)
        line(p_prev, p_cur)        // 회전 복제
        scale(1, -1)               // 거울
        line(p_prev, p_cur)        // 반사 복제
        pop()
```
캔버스를 매 프레임 지우지 않고 **누적**시키면 그림이 쌓인다.

## 주요 파라미터
- `n`(대칭 수, 기본 8): ←/→ 로 3~24 조절
- 좌표는 항상 **중심을 원점으로** 변환한 뒤 회전해야 한다.

## 다른 언어로 옮길 때
- 변환 스택이 없으면 각 점을 직접 회전: `x' = x·cosθ − y·sinθ`, `y' = x·sinθ + y·cosθ`. 반사는 `y → −y`.
- 획을 누적하려면 배경을 지우지 말거나, 별도 레이어(이미지 버퍼)에 그린다.
